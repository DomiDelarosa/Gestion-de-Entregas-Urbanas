# Gestion de Entregas Urbanas

Este proyecto modela un sistema de gestión de entregas urbanas. Permite administrar vehículos y pedidos, asignar rutas de distribución y optimizar la entrega mediante algoritmos intercambiables.

## Objetivos del sistema

  - Simular un sistema real de distribución urbana.
  - Evaluar la eficiencia de las rutas generadas. 
  - Servir como base para futuras mejoras y experimentación con distintos métodos de optimización.

## Métricas de evaluación

El sistema evalúa el funcionamiento del sistema mediante las siguientes métricas:
  - Distancia total recorrida.
  - Uso de capacidad de los vehículos
  - Cantidad de pedidos asignados.
  - Pedidos no atendidos.
